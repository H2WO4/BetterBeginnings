Mod Name

更好的初始牌组
---------------------------------------------
Mod Description

你是否有过这样的经历：早早就拿了精准/催化剂，却一整局见不到一张刀牌/毒牌的踪影？
你是否觉得前期游戏有点太依赖于最开始几次试探性的选牌？

那么这个mod就是为你准备的！

每个原版角色的初始牌组中都会有一张打击和一张防御替换成新的卡牌。
这些牌志在提供低层次的组合策略，以使得诸如精准之类的牌早早就适合抓取。


源代码可以在此处获取：https://github.com/H2WO4/BetterBeginnings
